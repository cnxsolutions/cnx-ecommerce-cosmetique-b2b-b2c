# Architecture – Plateforme E-commerce Terre d’Atlas

## Vue d’ensemble
La plateforme repose sur une architecture e-commerce
basée sur WordPress et WooCommerce, conçue pour répondre
aux besoins B2C et B2B, tout en restant évolutive et SEO-friendly.

## Composants principaux
- Frontend : thème WordPress optimisé SEO
- Backend : WooCommerce
- Base de données : MySQL
- Paiement : passerelles sécurisées Stripe | Paypal | Apple Pay 
- Logistique : génération automatique des documents et suivi colis

## Flux principaux
1. Consultation du catalogue par l’utilisateur
2. Passage de commande
3. Génération automatique :
   - facture
   - bon de livraison
4. Expédition et suivi

## Choix techniques & justification
- WooCommerce pour sa flexibilité B2C/B2B
- WordPress pour la rapidité de mise en œuvre
- Extensions ciblées pour limiter la complexité
